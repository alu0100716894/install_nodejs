Instalacion de nodejs en Ubuntu

Ponemos en consola los siguiente comandos por orden:

  1. sudo apt-get install curl
  2. curl -sL https://deb.nodesource.com/setup | sudo bash
  3. sudo apt-get install -y nodejs
